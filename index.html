HTML: A Estrutura da Web

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar páginas web. Ele define elementos como títulos, parágrafos, imagens e links.

Estrutura básica de um documento HTML

<!DOCTYPE html>
<html>
    <head>
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Título Principal</h1>
        <p>Este é um parágrafo.</p>
    </body>
</html>

Principais elementos do HTML

  • <h1> a <h6> - Títulos de diferentes níveis
  • <p> - Parágrafos
  • <a href="URL"> - Links
  • <img src="imagem.jpg"> - Imagens
  • <ul> e <li> - Listas não ordenadas

Curiosidade: Códigos de Entidade HTML

Em HTML, certos caracteres precisam ser escritos como entidades para serem exibidos corretamente. Aqui estão alguns exemplos:

  • < - `&lt;` (Menor que) → Exibe `<` sem ser interpretado como uma tag
  • > - `&gt;` (Maior que) → Exibe `>`
  • & - `&amp;` (Ampersand) → Exibe `&`
  • " - `&quot;` (Aspas duplas) → Exibe `"`
  • ' - `&apos;` (Aspas simples) → Exibe `'`

Essas entidades garantem que os caracteres sejam exibidos corretamente sem interferir na estrutura do código HTML.

CSS: Dando Estilo às Páginas Web

O CSS (Cascading Style Sheets) é usado para estilizar páginas web, permitindo que os desenvolvedores personalizem o visual de um site. É como a maquiagem ou o figurino de uma página, deixando-a bonita e funcional!

O que o CSS pode fazer?

  • Alterar as cores de fundo e do texto
  • Definir estilos de fontes (tamanho, tipo, espaçamento entre letras)
  • Criar layouts para organizar o conteúdo na tela
  • Adicionar efeitos visuais, como sombras e transições
  • Tornar o site responsivo, adaptando-o a diferentes tamanhos de tela

Como adicionar CSS ao HTML?

Existem três formas principais:

  • Estilo Inline: Estilo diretamente no elemento HTML. Exemplo: 
    <p style="color: red;">Texto vermelho</p>
  • Estilo Interno: Dentro de um bloco <style> no cabeçalho do HTML. Exemplo: 
            <style>
            body {
                background-color: lightblue;
            }
        </style>
  • Estilo Externo: Usando um arquivo separado, como styles.css. Exemplo: 
            /* Arquivo styles.css */
        h1 {
            color: green;
            font-family: Arial, sans-serif;
        }

Propriedades mais comuns do CSS:

Estas são algumas das propriedades CSS que você usará frequentemente:

Propriedade Descrição Exemplo
color Define a cor do texto color: blue;
background-color Define a cor de fundo background-color: yellow;
font-size Define o tamanho do texto font-size: 20px;
margin Espaço externo ao redor de um elemento margin: 10px;
padding Espaço interno entre o conteúdo e a borda padding: 15px;

Dicas para usar CSS como um profissional:

  • Use classes e ids para aplicar estilos específicos.
  • Comente o código para mantê-lo organizado e fácil de entender.
  • Use ferramentas como o "Inspecionar Elemento" do navegador para testar mudanças no CSS.
  • Aplique boas práticas de responsividade para que o site funcione bem em celulares e computadores.

Exemplo de Layout:

Um exemplo simples para estruturar uma página com CSS:

        /* Layout básico */
        header {
            background-color: navy;
            color: white;
            padding: 10px;
            text-align: center;
        }
        
        main {
            display: flex;
            justify-content: space-around;
            margin: 20px;
        }
        
        footer {
            background-color: navy;
            color: white;
            text-align: center;
            padding: 10px;
        }

JavaScript: Tornando as Páginas Interativas

O JavaScript é uma linguagem de programação que dá vida às páginas web. Ele permite que você crie ações dinâmicas, como responder a cliques de botão, exibir mensagens e interagir com o usuário.

Por que JavaScript é importante?

  • É essencial para o desenvolvimento web, sendo usado em praticamente todos os sites modernos.
  • Permite criar interatividade e experiências dinâmicas no navegador.
  • É fácil de começar e aprender, mas poderoso o suficiente para projetos complexos.

Como usar JavaScript?

JavaScript pode ser usado de várias maneiras:

  • Inline: Adicionando código diretamente no elemento HTML (não é recomendado para projetos maiores). 
    <button onclick="alert('Olá, mundo!')">Clique aqui</button>
  • Interno: Dentro de uma tag <script> no arquivo HTML. 
            <script>
            console.log("Bem-vindo ao JavaScript!");
        </script>
  • Externo: Utilizando um arquivo separado, como script.js. 
            // Arquivo script.js
        function mostrarMensagem() {
            alert("Você clicou no botão!");
        }
                    
            <button onclick="mostrarMensagem()">Clique aqui</button>

Conceitos básicos:

  • Variáveis: Armazenam valores que podem ser usados no código. 
            let nome = "João";
        console.log(nome); // Exibe "João"
  • Funções: Executam uma tarefa específica. 
            function somar(a, b) {
            return a + b;
        }
        console.log(somar(2, 3)); // Exibe 5
  • Eventos: Respondem às ações do usuário, como cliques. 
            document.getElementById("meuBotao").addEventListener("click", function() {
            alert("Botão clicado!");
        });

Manipulação do DOM:

O DOM (Document Object Model) permite que o JavaScript interaja com os elementos HTML.

        document.getElementById("meuTexto").innerHTML = "Texto alterado!";

HTML relacionado:

        <p id="meuTexto">Texto original</p>
        <button onclick="document.getElementById('meuTexto').innerHTML = 'Texto alterado!'">Alterar Texto</button>
